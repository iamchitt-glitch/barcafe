🔗 Apps Script URL
URL นี้ใช้ร่วมกันได้ทั้งหน้าเจ้าของและหน้าพนักงาน
📋 วิธีตั้งค่า Apps Script (ทำครั้งเดียว)
  1. เปิด Google Sheets ใหม่ ตั้งชื่อ "BAR CAFE Stock"
  2. Extensions → Apps Script → ลบโค้ดเดิม → วางโค้ดจากไฟล์ Code.gs → Save
  3. Deploy → New deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Deploy
  4. Copy URL → วางในช่องด้านบน → บันทึก
✓ ข้อมูลจาก Google Doc 20 มีนา จะโหลดเข้า Sheet อัตโนมัติในครั้งแรก ไม่ต้องคีย์ใหม่
